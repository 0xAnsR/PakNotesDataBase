# PakNotesDataBase

PakNotesDataBase is a local repository for preparing and organizing test-related English MCQ data.

## Current Workspace

This repository is now connected to GitHub and ready for live edits, commits, and pushes from the local Codex workspace.

## Notes

- `imports/` contains generated import pages and is ignored by Git.
- `__pycache__/` contains Python cache files and is ignored by Git.
- `data/DB COPY/` is ignored so duplicate database backups are not pushed by accident.
